## Что где лежит

`task_1`

- каталог под решение первой части

`task_2_1_api`

- автоматизированные API-тесты для сервиса объявлений
- код, тесты, документация и конфиги разложены по отдельным подкаталогам
- тест-кейсы, баг-репорты, инструкции по запуску и конфигурация инструментов качества кода

## Кратко по второму заданию

Во втором задании реализованы:

- позитивные, негативные и corner-case проверки
- E2E-сценарии
- Allure-аннотации (отчет можно посмотреть [здесь](task_2_1_api/docs/ALLURE.md))
- линтер и базовая конфигурация качества кода
- документация по запуску и найденным дефектам

Подробности, команды запуска и описание файлов находятся в [README второго задания](./task_2_1_api/README.md).
