# Баги по скриншоту

 Ниже перечислены только те проблемы, которые видны визуально.

## P1 — высокий

1. **Не работает фильтр цены.**
   - В фильтре задан диапазон: **от 1 000 до 30 000 ₽**.
   - В выдаче присутствуют карточки с ценой **500 ₽** и **40 000 ₽**.

2. **Не работает фильтр пола.**
   - Активен фильтр **«Мальчик»**.
   - В выдаче есть карточка **«Персидский котёнок девочка»**.

3. **Не работает фильтр локации.**
   - В шапке страницы и в гео указана **Москва**.
   - В выдаче есть объявление с локацией **Новосибирск**.

4. **Не работает фильтр породы.**
   - В фильтре выбрана порода **«Персидская»**.
   - В выдаче есть карточка **«Экзотический короткошерстный котёнок»**.

5. **В выдачу живых животных попал нерелевантный товар.**
   - Страница открыта в категории **«Животные → Кошки»**.
   - В выдаче отображается карточка **«Персидский котёнок плюшевый»**, которая является игрушкой, а не живое животным.

## P2 — средний

6. **На странице одновременно показан контент и системная ошибка.**
   - Справа сверху отображается сообщение **«Ошибка, обновите страницу»**.
   - При этом сама выдача с карточками уже загружена и отображается.
   - Это некорректное состояние интерфейса или необработанная ошибка частичной загрузки.

7. **Использована неверная терминология в фильтре возраста.**
   - Страница относится к категории кошек / котят.
   - В фильтре возраста используется значение **«Щенок»** вместо ожидаемого **«Котёнок»**.

8. **Задано неверное описание кнопки.**
   - Подсказка **«Не показывайте мне похожие объявления»** не соответствует внешнему виду.

9. **Нарушение геометрии сетки карточек.**
   - Карточки первого ряда имеют идентичные отступы слева и справа.
   - У второй карточки второго ряда отступ слева значительно больше, чем у остальных.

10. **Неконсистентные счетчики результатов.**
    - В заголовке страницы указано **19** объявлений.
    - На кнопке в левой колонке написано **«Показать 8 объявлений»**.
    - Без дополнительных пояснений это выглядит как расхождение между количеством результатов на странице и количеством после примененных фильтров.

## P3 — низкий

11. **Дублирование / пересечение фильтров по типу продавца.**
    - В блоке **«Вид объявления»** есть опция **«От частного владельца»**.
    - Ниже есть отдельный блок **«Продавцы»** с опцией **«Частные»**.
    - Для пользователя это выглядит как дублирование одного и того же критерия в разных местах.

12. **Неочевидная гео-логика в сортировке / переключателе.**
    - При выбранной локации **Москва** отображается переключатель **«Сначала из Можайска»**.
    - Без выбора конкретного района это выглядит неожиданно и может указывать на некорректно подставленный контекст.

13. **Нелогичная иерархия категории объявления.**
    - В breadcrumb отображается цепочка **«Главная > Животные > Персидская > Кошки»**.
    - Порода показана как уровень навигации между категорией и подкатегорией.
